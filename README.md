# NekoBridge 搭路练习插件

## 简介
NekoBridge 是一个专为Minecraft搭路练习设计的插件，提供了完整的搭路练习环境，包括速度统计、自动复活、无限方块等功能。

## 功能特性

### 基础功能
- **移除消息**：自动移除玩家加入、退出和死亡消息
- **无限沙石**：玩家始终拥有64个沙石方块
- **效率镐**：提供效率III且无限耐久的钻石镐
- **速度统计**：实时显示玩家搭路速度（方块/秒）

### 出生点系统
- **全局出生点**：使用 `/setspawn` 指令设置全局出生点（仅OP可用）
- **个人出生点**：玩家踩到绿宝石块时自动设置为个人出生点
- **完成点**：玩家踩到红石块时传送回出生点并清理所有方块

### 方块管理
- **方块限制**：只允许玩家破坏自己放置的方块（创造模式除外）
- **自动清理**：玩家死亡或退出时自动清理其放置的方块
- **破坏音效**：破坏方块时播放音效，按顺序逐步破坏

### 其他特性
- **锁定天气**：所有世界锁定为晴天
- **死亡保护**：玩家死亡后自动复活并传送回出生点
- **无掉落物**：死亡和破坏方块时无掉落物

## 安装方法

1. 将 `NekoBridge-1.0-SNAPSHOT.jar` 文件放入服务器的 `plugins` 文件夹
2. 重启服务器
3. 插件会自动生成配置文件

## 指令

| 指令 | 权限 | 说明 |
|------|------|------|
| `/setspawn` | `neko.bridge.setspawn` | 设置全局出生点（仅OP） |

## 方块功能

| 方块类型 | 功能 |
|----------|------|
| 绿宝石块 | 设置个人出生点 |
| 红石块 | 完成点（传送回出生点并清理方块） |

## 配置文件

插件会在首次运行时生成 `config.yml` 文件，包含全局出生点设置。

## 版本兼容性

- Minecraft 1.12.2
- Paper/Spigot 1.12.2

## 注意事项

- 请确保服务器已安装兼容的Minecraft版本
- 玩家需要在生存或创造模式下才能获得沙石和钻石镐
- 插件会自动锁定天气为晴天
